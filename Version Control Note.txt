Version Control Technology
=============================
GIT
1. installation
Git SCM installation on a local machine


2. Link up local server to remote or get a copy of remote

How to get remote repsoitory to talk to local repository:
	1. get a copy of the remote serve/repo unto the local lappy => cloning 
>git clone https://team-msc-2018.visualstudio.com/mscsharp.programs.repo/_git/mscsharp.programs.repo

	2. add the address of the remote repo to the local
>git remote add origin https://team-msc-2018.visualstudio.com/mscsharp.programs.repo/_git/mscsharp.programs.repo

How to check-in codes in the Git server (VSTS)
==============================================
Assuming you have already cloned a repository
1. Navigate inside of the repo and open up and batch
	$ git status
	$ git add --all(git add filename/git add ./ if a file name has sapces in it => git add 'file name')
	$ git commit -m "Message to describe whatever you have done"
	$ git fetch && git pull origin master (in case someone check-in before you)
	$ git push origin master

==============================================
